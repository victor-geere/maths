# Root Systems & Dynkin Diagrams

A **root system** is a finite set of nonzero vectors (roots) in a Euclidean space satisfying strict geometric axioms about reflections and integrality. Root systems are the combinatorial skeleton of semisimple Lie algebras: the **roots** are the nonzero eigenvalues of the adjoint action of a maximal abelian subalgebra (a Cartan subalgebra), and they control the entire structure and representation theory of the algebra. The classification of irreducible root systems by **Dynkin diagrams** is one of the most elegant results in mathematics, yielding exactly the four infinite families ($A_n, B_n, C_n, D_n$) plus five exceptional systems ($E_6, E_7, E_8, F_4, G_2$).

## Definition

A **root system** in a Euclidean space $(V, \langle\cdot,\cdot\rangle)$ is a finite set $\Phi \subset V \setminus \{0\}$ satisfying:
1. $\Phi$ spans $V$.
2. If $\alpha \in \Phi$ then $-\alpha \in \Phi$ (and $\mathbb{R}\alpha \cap \Phi = \{\pm\alpha\}$).
3. For all $\alpha, \beta \in \Phi$: $s_\alpha(\beta) = \beta - \frac{2\langle\beta,\alpha\rangle}{\langle\alpha,\alpha\rangle}\alpha \in \Phi$ (closure under reflections).
4. **Integrality**: $\frac{2\langle\beta,\alpha\rangle}{\langle\alpha,\alpha\rangle} \in \mathbb{Z}$ for all $\alpha, \beta \in \Phi$.

## Cartan Matrix & Dynkin Diagram

Choose a set of **simple roots** $\{\alpha_1,\ldots,\alpha_r\}$ (a basis of $V$ in $\Phi$). The **Cartan matrix** is $A_{ij} = \frac{2\langle\alpha_i,\alpha_j\rangle}{\langle\alpha_j,\alpha_j\rangle}$. The **Dynkin diagram** encodes $A$: each simple root is a node; nodes $i,j$ are connected by $|A_{ij}||A_{ji}|$ edges (with an arrow when $|\alpha_i| \neq |\alpha_j|$).

## Classification

Connected Dynkin diagrams:
- **$A_n$**: $SU(n+1)$ — chain of $n$ nodes
- **$B_n$**: $SO(2n+1)$ — chain with one double arrow at end
- **$C_n$**: $Sp_{2n}$ — chain with reversed double arrow
- **$D_n$**: $SO(2n)$ — chain with forked end
- **Exceptionals**: $E_6, E_7, E_8, F_4, G_2$

## Glossary

| Term / Symbol | Meaning |
|---|---|
| Root system $\Phi$ | finite set of vectors in $(V,\langle\cdot,\cdot\rangle)$ satisfying four axioms |
| Root $\alpha$ | element of $\Phi$; eigenvalue of adjoint Cartan action |
| Simple roots $\alpha_1,\ldots,\alpha_r$ | basis of $V$ from roots; all other roots are $\mathbb{Z}_+$-combinations |
| Rank $r$ | dimension of $V$ |
| Reflection $s_\alpha$ | $\beta \mapsto \beta - \frac{2\langle\beta,\alpha\rangle}{|\alpha|^2}\alpha$ |
| Cartan matrix $A_{ij}$ | $\frac{2\langle\alpha_i,\alpha_j\rangle}{\langle\alpha_j,\alpha_j\rangle}$; integer-valued |
| Dynkin diagram | graph encoding Cartan matrix; classifies root systems |
| $A_n, B_n, C_n, D_n$ | four infinite families of classical Lie algebras |
| $E_6,E_7,E_8,F_4,G_2$ | five exceptional root systems |
| Weyl group $W$ | group generated by reflections $s_\alpha$; finite subgroup of $O(V)$ |
