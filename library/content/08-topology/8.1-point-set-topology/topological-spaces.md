# Topological Spaces

A **topological space** is the most general setting in which it makes sense to talk about **continuity**, **convergence**, and **connectedness** — without requiring a distance function. The structure consists of a set $X$ together with a designated collection $\tau$ of subsets (the **open sets**), subject to just three axioms. This abstraction unifies metric spaces, manifolds, algebraic varieties, and combinatorial complexes into a single framework, and allows theorems proved in this generality to apply to all of them at once. Topology is sometimes called "rubber-sheet geometry" because it studies properties that survive continuous deformation — stretching and bending but not tearing or gluing.

## Definition

A **topological space** is a pair $(X, \tau)$ where $X$ is a set and $\tau \subseteq \mathcal{P}(X)$ is a **topology** satisfying:

1. $\emptyset \in \tau$ and $X \in \tau$
2. Arbitrary unions: if $U_\alpha \in \tau$ for all $\alpha$, then $\bigcup_\alpha U_\alpha \in \tau$
3. Finite intersections: if $U_1, \ldots, U_n \in \tau$, then $U_1 \cap \cdots \cap U_n \in \tau$

The sets in $\tau$ are the **open sets** of $(X, \tau)$.

## Standard Examples

| Space | Topology $\tau$ |
|---|---|
| Any metric space $(X,d)$ | metric topology (open balls generate it) |
| $\mathbb{R}$ | standard topology (open intervals) |
| Discrete: any set $X$ | all subsets of $X$ are open |
| Indiscrete: any set $X$ | only $\emptyset$ and $X$ are open |
| Finite complement topology | open $=$ sets with finite complement, plus $\emptyset$ |

## Comparing Topologies

If $\tau_1 \subseteq \tau_2$ on $X$: $\tau_1$ is **coarser** (fewer open sets) and $\tau_2$ is **finer** (more open sets).

- Indiscrete is the coarsest; discrete is the finest topology on any set.

## Subspace Topology

If $A \subseteq X$, the **subspace topology** on $A$ is $\tau_A = \{U \cap A : U \in \tau\}$.

## Product Topology

For spaces $(X, \tau_X)$ and $(Y, \tau_Y)$, the **product topology** on $X \times Y$ has basis $\{U \times V : U \in \tau_X, V \in \tau_Y\}$.

## Homeomorphism

A **homeomorphism** is a continuous bijection whose inverse is also continuous. Homeomorphic spaces are topologically identical — a coffee mug and a donut are homeomorphic.

## Glossary

| Term / Symbol | Meaning |
|---|---|
| $(X, \tau)$ | a topological space: set $X$ with topology $\tau$ |
| $\tau$ (tau) | topology — the designated collection of open sets |
| $\mathcal{P}(X)$ | power set of $X$ (all subsets) |
| Open set | a member of $\tau$ |
| Discrete topology | every subset is open; finest possible topology |
| Indiscrete topology | only $\emptyset$ and $X$ are open; coarsest possible |
| Coarser topology | fewer open sets |
| Finer topology | more open sets |
| Subspace topology | open sets of $A$ are intersections of open sets of $X$ with $A$ |
| Product topology | topology on $X \times Y$ generated by products of open sets |
| Homeomorphism | continuous bijection with continuous inverse |
| Topological property | one preserved by homeomorphism (e.g. compactness, connectedness) |
