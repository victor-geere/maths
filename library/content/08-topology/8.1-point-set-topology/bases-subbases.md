# Bases & Subbases

Specifying a topology by listing every open set is often impractical — a topology on $\mathbb{R}$ contains uncountably many open sets. A **basis** solves this by providing a smaller generating family: every open set is a union of basis elements, so the basis completely determines the topology but is far simpler to describe. A **subbasis** is even more economical: we take arbitrary finite intersections of subbasis elements to form a basis, then take unions. These concepts are essential for constructing product topologies, comparing topologies, and working with specific spaces like $\mathbb{R}^n$ where the open balls form a natural basis.

## Basis

A collection $\mathcal{B} \subseteq \tau$ is a **basis** for a topology $\tau$ on $X$ if every open set $U \in \tau$ can be written as a union of elements of $\mathcal{B}$.

**Basis axioms** (for generating a topology from scratch): $\mathcal{B}$ is a basis iff:
1. $\bigcup_{B \in \mathcal{B}} B = X$ (covers $X$)
2. If $x \in B_1 \cap B_2$ (with $B_1, B_2 \in \mathcal{B}$), then $\exists\, B_3 \in \mathcal{B}$ with $x \in B_3 \subseteq B_1 \cap B_2$

The **generated topology** is $\tau = \{\bigcup \mathcal{B}' : \mathcal{B}' \subseteq \mathcal{B}\}$.

## Standard Bases

| Space | Natural basis |
|---|---|
| $\mathbb{R}$ (standard) | open intervals $(a, b)$ |
| $\mathbb{R}^n$ | open balls $B(x, r)$ |
| $\mathbb{R}^n$ | open boxes $(a_1,b_1)\times\cdots\times(a_n,b_n)$ |
| Discrete topology on $X$ | singletons $\{x\}$ |

## Subbasis

A collection $\mathcal{S} \subseteq \mathcal{P}(X)$ is a **subbasis** if the set of all finite intersections of members of $\mathcal{S}$ forms a basis.

The topology generated by $\mathcal{S}$ is the **coarsest** topology containing $\mathcal{S}$.

## Product Topology via Subbasis

The product topology on $\prod_\alpha X_\alpha$ is generated by the subbasis:

$$\mathcal{S} = \{\pi_\alpha^{-1}(U_\alpha) : U_\alpha \text{ open in } X_\alpha\}$$

where $\pi_\alpha$ is the projection onto the $\alpha$-th factor. This is the coarsest topology making all projections continuous.

## Second Countability

A space is **second countable** if it has a countable basis. Second countable spaces are separable (have a countable dense subset) and satisfy many useful properties.

## Glossary

| Term / Symbol | Meaning |
|---|---|
| $\mathcal{B}$ | basis — a generating family for a topology |
| $\mathcal{S}$ | subbasis — an even smaller generating family |
| Generated topology | the coarsest topology containing a given collection |
| $\bigcup \mathcal{B}'$ | union of all sets in a sub-collection $\mathcal{B}'$ |
| Finite intersection | intersection of finitely many sets |
| $\pi_\alpha$ | projection onto the $\alpha$-th coordinate in a product |
| $\pi_\alpha^{-1}(U_\alpha)$ | the preimage of $U_\alpha$ under $\pi_\alpha$ — a "cylinder" set |
| Product topology | coarsest topology making all projections continuous |
| Second countable | admits a countable basis; e.g. $\mathbb{R}^n$ |
| Separable | has a countable dense subset; e.g. $\mathbb{R}$ (with $\mathbb{Q}$ dense) |
| Dense subset | $D$ is dense in $X$ if $\overline{D} = X$ |
