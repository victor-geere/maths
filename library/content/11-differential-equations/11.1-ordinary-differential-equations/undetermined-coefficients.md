# Method of Undetermined Coefficients

The **method of undetermined coefficients** finds a particular solution $y_p$ to a non-homogeneous linear ODE with constant coefficients by guessing that $y_p$ has the same form as the forcing function $f(x)$. If $f(x)$ is a polynomial, exponential, sine, cosine, or a product of these, the appropriate trial function is a linear combination of all terms generated by repeatedly differentiating $f(x)$. The trial function is substituted into the ODE, and the coefficients are determined algebraically. This method is efficient when it applies, but it is restricted to forcing functions that produce finite families under differentiation.

## When to Use

The method works when $f(x)$ is one of (or a product of):
- Polynomials: $x^n$
- Exponentials: $e^{ax}$
- Sines/cosines: $\sin\beta x$, $\cos\beta x$
- Products thereof: $x^2 e^{3x}\cos 2x$, etc.

## Trial Function Table

| $f(x)$ | Trial $y_p$ |
|---|---|
| $P_n(x)$ (degree-$n$ poly) | $A_nx^n + \cdots + A_1x + A_0$ |
| $e^{ax}$ | $Ae^{ax}$ |
| $\sin\beta x$ or $\cos\beta x$ | $A\cos\beta x + B\sin\beta x$ |
| $e^{ax}\sin\beta x$ | $e^{ax}(A\cos\beta x + B\sin\beta x)$ |
| $P_n(x)e^{ax}$ | $e^{ax}(A_nx^n + \cdots + A_0)$ |

## Modification Rule

If any term of the trial function solves the homogeneous equation, **multiply the entire trial function by $x$** (or $x^2$ if necessary to clear all resonances).

## Example

$y'' - 3y' + 2y = e^{3x}$. Homogeneous roots: $r = 1, 2$.

Trial: $y_p = Ae^{3x}$. Substitute: $9Ae^{3x} - 9Ae^{3x} + 2Ae^{3x} = 2Ae^{3x} = e^{3x}$, so $A = 1/2$.

$y_p = \tfrac{1}{2}e^{3x}$.

**Modification example:** $y'' - 3y' + 2y = e^{x}$. Since $e^x$ solves the homogeneous equation ($r=1$), use $y_p = Axe^x$.

## Glossary

| Term / Symbol | Meaning |
|---|---|
| $y_p$ | particular solution found by this method |
| Undetermined coefficients | unknown constants $A, B, \ldots$ in the trial function |
| Trial function | the guessed form of $y_p$ based on $f(x)$ |
| Forcing function $f(x)$ | the right-hand side of the non-homogeneous ODE |
| Modification rule | multiply trial by $x^k$ when the trial solves the homogeneous equation |
| Resonance | occurs when $f(x)$ shares a term with $y_h$; requires modification |
| $P_n(x)$ | a general polynomial of degree $n$ |
| $y_h$ | homogeneous solution (complementary function) |
| General solution | $y = y_h + y_p$ |
| Constant coefficients | the ODE coefficients are constants, not functions of $x$ |
