## Definition

$$|x| = \begin{cases} x & x \geq 0 \\ -x & x < 0 \end{cases}$$

Geometrically, $|x|$ is the distance from $x$ to $0$ on the number line.

## Key Property

$$|x| = c \iff x = c \text{ or } x = -c \quad (c \geq 0)$$

## Solving Absolute Value Equations

**Example:** $|2x - 3| = 7$

$$2x - 3 = 7 \implies x = 5 \qquad \text{or} \qquad 2x - 3 = -7 \implies x = -2$$

## Absolute Value Inequalities

| Form | Equivalent |
|---|---|
| $|x| < c$ | $-c < x < c$ |
| $|x| > c$ | $x < -c$ or $x > c$ |
| $|x| \leq c$ | $-c \leq x \leq c$ |
| $|x| \geq c$ | $x \leq -c$ or $x \geq c$ |

**Example:** $|x - 1| < 4$

$$-4 < x - 1 < 4 \implies -3 < x < 5$$

## Triangle Inequality

$$|a + b| \leq |a| + |b|$$

Equality holds iff $a$ and $b$ have the same sign.

## Notes

- $|x|^2 = x^2$ for all $x \in \mathbb{R}$.
- $|xy| = |x||y|$ and $\left|\tfrac{x}{y}\right| = \tfrac{|x|}{|y|}$ for $y \neq 0$.
