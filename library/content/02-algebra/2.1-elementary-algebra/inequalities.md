## Basic Rules

For real numbers $a, b, c$:

- If $a < b$ and $c > 0$, then $ac < bc$.
- If $a < b$ and $c < 0$, then $ac > bc$ (**inequality flips**).
- If $a < b$ and $b < c$, then $a < c$ (transitivity).

## Interval Notation

| Inequality | Interval |
|---|---|
| $a < x < b$ | $(a, b)$ |
| $a \leq x \leq b$ | $[a, b]$ |
| $x > a$ | $(a, \infty)$ |
| $x \leq b$ | $(-\infty, b]$ |

## Solving Linear Inequalities

Treat like an equation but **flip the sign when multiplying/dividing by a negative**.

**Example:** $-2x + 3 > 7$

$$-2x > 4 \implies x < -2$$

Solution: $(-\infty, -2)$.

## Solving Quadratic Inequalities

Find roots, then test intervals.

**Example:** $x^2 - x - 6 > 0 \Rightarrow (x-3)(x+2) > 0$

Roots: $x = -2, 3$. Test intervals: $(-\infty,-2)$, $(-2,3)$, $(3,\infty)$.

Sign positive on $(-\infty,-2) \cup (3,\infty)$.

## Compound Inequalities

- **And** ($\cap$): $1 < x$ and $x < 5$ → $(1, 5)$.
- **Or** ($\cup$): $x < -1$ or $x > 3$ → $(-\infty,-1) \cup (3,\infty)$.

## Polynomial Inequalities

Use a **sign chart**: factor the polynomial, identify roots, and determine the sign on each interval.
