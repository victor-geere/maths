Overview

The dyadic prime generator constructs all composites in each interval \(I_n = [2^n, 2^{n+1}-1]\) using the bound \(\Omega(m) \leq n\). Primes appear as gaps. Its spectral properties reveal a beautiful interplay between arithmetic constraints (parity) and analytic structure (Riemann zeros).

This document synthesizes the full theoretical development, closed forms, generalizations, and deep spectral analysis with live interactive visualizations.

Core Objects

  • Generator: \(G_n = \{ m \in I_n \mid \tau(m) \geq 3 \}\)
  • Primes: Gaps in \(G_n\)
  • Spectral views: DFT inside shells + Mellin across shells

Decay of Nyquist Relative Power

Relative contribution of the period-2 spike decays as ~1/n, as predicted by prime density.

Key Spectral Findings

  • Persistent Nyquist Spike: Always at normalized frequency 0.5. Power = (#Primes)² exactly. Caused by all even numbers >2 being composite.
  • Background Spectrum: After removing the spike, the spectrum is consistent with a thinned random process on odd integers.
  • Low-frequency modulations: Small contributions from low-lying Riemann zeros appear as excess power in the first few bins.

Riemann Zeta Function Symmetries

The completed zeta function satisfies \(\xi(s) = \xi(1-s)\). This implies:

  • Conjugate symmetry: zeros come in conjugate pairs.
  • Reflection symmetry: if \(\rho\) is a zero then so is \(1-\rho\).

In the explicit formula these symmetries produce paired oscillatory contributions to prime counts in dyadic shells. The Mellin transform across shells inherits a symmetry under \(s \to 1-s\).

Impact on Generator

The gaps in \(G_n\) inherit symmetric spectral signatures. Oscillations in low-frequency DFT bins and across \(n\) respect the functional equation symmetry, making the analytic layer of the spectrum invariant under reflection.

Pair Correlation of Zeta Zeros

Montgomery’s pair correlation conjecture states that the normalized spacings between imaginary parts of zeros follow GUE (Gaussian Unitary Ensemble) statistics from random matrix theory.

Level Repulsion

Probability of two zeros being extremely close is quadratic in separation — no clustering at zero distance.

Rigidity

Zeros are more regularly spaced than a Poisson process, leading to more uniform prime gap distributions at certain scales.

Connection to Gaps

Via the explicit formula, pair correlation controls the fine structure of prime gaps inside each \(I_n\) after the dominant arithmetic layer is removed.

In the generator, pair correlation influences the higher-frequency components of the DFT (after subtracting the Nyquist spike) and the statistics of gap sizes within large shells.

Generalizations

Global Function Fields

Replace magnitude with degree. Generator on exact-degree shells. Gaps = prime divisors of degree n. Exact counts via zeta function of the curve. Riemann hypothesis proven (Weil). Pair correlation analogs proven (Katz–Sarnak).

Number Fields (Ideals)

Use norm as size. Generator on ideals with norm in dyadic shells. Gaps = prime ideals in the shell. Dedekind zeta function governs oscillations. Class group adds extra structure.

Python Implementation

Here is a clean, practical Python implementation of the generator. It uses a segmented sieve approach (the efficient realization of the product-based generator) to find all primes (gaps) in \(I_n\).

import math

def generate_primes_in_dyadic_interval(n):
    """
    Find all primes in the dyadic interval I_n = [2^n, 2^{n+1} - 1]
    using a segmented sieve (efficient implementation of the generator).
    """
    if n < 1:
        return []
    
    L = 1 << n                    # lower bound = 2^n
    U = (1 << (n + 1)) - 1        # upper bound = 2^{n+1} - 1
    
    # Step 1: Generate all small primes up to sqrt(U)
    sqrt_U = int(math.sqrt(U)) + 1
    is_small_prime = [True] * (sqrt_U + 1)
    is_small_prime[0] = is_small_prime[1] = False
    
    for i in range(2, int(math.sqrt(sqrt_U)) + 1):
        if is_small_prime[i]:
            for j in range(i * i, sqrt_U + 1, i):
                is_small_prime[j] = False
    
    small_primes = [i for i in range(2, sqrt_U + 1) if is_small_prime[i]]
    
    # Step 2: Sieve the interval [L, U]
    length = U - L + 1
    is_prime = [True] * length
    
    # Handle small cases
    if L == 0:
        is_prime[0] = False
    if L <= 1 <= U:
        is_prime[1 - L] = False
    
    for p in small_primes:
        if p * p > U:
            break
        
        # Find the first multiple of p that is >= L
        start = max(p * p, ((L + p - 1) // p) * p)
        
        for multiple in range(start, U + 1, p):
            if multiple >= L:
                is_prime[multiple - L] = False
    
    # Collect primes
    primes = [L + i for i in range(length) if is_prime[i]]
    
    # Ensure 2 is handled correctly if in range
    if L <= 2 <= U and 2 not in primes:
        primes.insert(0, 2)
    
    return sorted(primes)


# Example usage
if __name__ == "__main__":
    n = 8
    primes = generate_primes_in_dyadic_interval(n)
    print(f"Primes in I_{n} = [2^{n}, 2^{n+1}-1]:")
    print(primes)
    print(f"\nTotal primes found: {len(primes)}")
This implementation directly realizes the generator: it uses all primes up to \(\sqrt{2^{n+1}}\) (available from previous layers) to mark composites in \(I_n\). The unmarked numbers are the primes (gaps in \(G_n\)).
